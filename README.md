# personal-api-gateway
Personal API Gateway (POC only, not for production)

### Introduction
Personal API Gateway is a Proof-Of-Concept (POC) project on how to implement API gateway from scratch.
Personal API Gateway is a **Centralized Edget Gateway** which uses **Hexagonal Architecture**.

### Features
1. Request Routing
2. Request & Response Transformation (Allow & Deny Headers)
3. Logging & Monitoring
4. Static Content Caching
5. Rate Limiter
6. Simple Load Balancing
